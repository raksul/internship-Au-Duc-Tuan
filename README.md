## Installations

```bash
cd smartphones-shop
# install dependencies
$ npm install

# serve with hot reload at localhost:3001
$ json-server --watch db.json
$ PORT=3001 npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
